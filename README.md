# Medidas Descritivas

Projeto desenvolvido no segundo semestre do curso de Engenharia de Software para a disciplina de Estatística

## ⚙ Funcionalidades

Ao inserir os números desejados o programa exibe:
- Média Aritmética
- Mediana
- Desvio Padrão
- Variância Amostral
- Coeficiente de Variação 
- Conjuntos de Dados


## 🛠 Tecnologias 

- HTML

- CSS

- JavaScript
